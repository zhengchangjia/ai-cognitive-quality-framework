# ai-cognitive-quality-framework
AI认知质量评估框架 — 评估AI分析报告的多维度体系，涵盖认知增量、逻辑一致性、思维严谨度等9个维度，区分“安全平庸”与“有洞见但存在瑕疵”的报告。AI Cognitive Quality Framework — A multi-dimensional evaluation system for AI-generated analytical reports across 9 dimensions including cognitive increment, logical consistency, and analytical rigor.
