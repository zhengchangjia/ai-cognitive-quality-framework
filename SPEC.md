# AI认知质量评估
## 评估标准 V6.4 — 全维度等级制评分细则

**核心原则（全维度通用）**：
- **0分**：该项工作完全没有做，或仅是复述素材内容，无任何认知增值。
- **1-3分**：表面工作，有动作但无实质产出。
- **4-6分**：基础完成，有产出但深度有限。
- **7-9分**：显著贡献，产出质量高，构成报告的真正价值。
- **10分**：卓越，该维度的理想范本。

**“仅是复述素材”的操作性定义**：将素材信息用不同措辞重新表述，读者读完未获得任何仅读素材无法直接获得的理解。

**全维度通用否决规则**：任一维度中，若有 **3 个及以上子维度得分 ≤ 3 分**，该维度总分直接归零，备注“多个子维度表现极差，维度归零”。

---

## 维度一：认知增量（权重：0.20）

**操作性定义（全维度通用）**：认知增量检测的是报告是否提供了超越素材复述的新洞察。

### 子维度1.1：新框架/概念构建（权重：0.25）

**操作性定义**：检测报告是否提出了素材中没有直接表述、但能重新组织多个现象之间关系的分析框架或概念。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 无框架 | 没有任何框架或概念，按素材原有结构复述。 |
| **1-3** | 表层整理 | 对素材做了分类或分段，但分类维度是素材自带的（如按利益方分组）。不构成新框架。 |
| **4-6** | 有限概念化 | 提出了局部概念或二分法标签，但不够系统，不足以统摄全局。 |
| **7-9** | 结构性框架 | 提出了素材中没有、能重新组织多个现象之间关系的分析框架，解释了“为什么多个现象同时发生”。 |
| **10** | 原创性概念 | 提出了可迁移、可泛化的理论概念，解释力超出当前案例。 |

### 子维度1.2：隐藏模式/机制揭示（权重：0.25）

**操作性定义**：检测报告是否发现了素材中不同信息之间的隐藏关联、深层动机或系统性机制。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 无揭示 | 未揭示任何机制，仅复述素材明说的因果。 |
| **1-3** | 显性归因 | 串联了因果链条，但链条的每个环节素材已明说。实质是把素材的因果语言重组了一遍。 |
| **4-6** | 局部机制发现 | 揭示了一两个素材未明说但可推断的动机或机制，属于单点洞察。 |
| **7-9** | 系统性机制揭示 | 揭示了跨主体、跨层级的隐性博弈结构或反馈环路，解释了表面矛盾行为背后的统一逻辑。 |
| **10** | 深层结构揭示 | 揭示了反直觉的结构性矛盾，可重释全局现象。 |

### 子维度1.3：可操作洞见产出（权重：0.20）

**操作性定义**：检测报告是否提供了基于分析的可操作判断、决策建议或风险评估。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 无洞见 | 以总结收尾，未提供可操作的判断或建议。 |
| **1-3** | 模糊方向 | 提供了方向性判断但不具体、不可检验。如“需要在X和Y间找到平衡”。 |
| **4-6** | 有条件的判断 | 提供了可检验的“如果-那么”判断，附带条件或边界。 |
| **7-9** | 具体策略建议 | 提供了带时间窗口、风险等级或行动路径的具体建议，可据此决策。 |
| **10** | 战略级洞见 | 提供了改变决策框架的洞见，附带验证路径与失效条件。 |

### 子维度1.4：提问/假设生成（权重：0.15）

**操作性定义**：检测报告是否提出了素材未覆盖但值得追问的问题，或生成了可检验的假设。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 无提问 | 未提出任何问题或假设。 |
| **1-3** | 修辞性追问 | 提出了问题但答案已在素材中，或问题过于空泛无法检验。 |
| **4-6** | 可检验假设 | 提出一个具体的、可被数据证实或证伪的假设。 |
| **7-9** | 多条追问链 | 衍生出多个可检验假设，并标注验证路径与不确定性级别。 |
| **10** | 重构问题框架 | 提出的问题改变了理解事件的框架本身，揭示了更深层的待解难题。 |

### 子维度1.5：复述规避度（权重：0.15）

**操作性定义**：检测报告中纯复述素材已有内容的篇幅占比。

| 分数 | 等级 | 行为描述（纯复述篇幅占比） |
|:---|:---|:---|
| **0** | 完全复述 | > 90% |
| **1-3** | 高度复述 | 70-90% |
| **4-6** | 混杂 | 40-70% |
| **7-9** | 分析主导 | 15-40% |
| **10** | 完全增值 | < 15% |

**否决条件**：
- 若1.1和1.2同时 ≤ 3分，维度得分直接归零。备注：报告未提供超越素材复述的认知增量。
- 若该维度有3个及以上子维度得分 ≤ 3分，维度得分直接归零。备注：多个子维度表现极差，维度归零。

## 维度二：外部逻辑一致性（权重：0.10）

**操作性定义（全维度通用）**：外部逻辑一致性检测的是报告的推论与素材中明确陈述的事实之间是否存在冲突。

### 子维度2.1：推论与事实的矛盾数量（权重：0.40）

**操作性定义**：检测报告的核心推论是否与素材中明确陈述的事实相悖。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 严重矛盾 | 核心结论直接与素材事实相悖。 |
| **1-3** | 多处小矛盾 | 存在2处以上局部推论与素材事实不一致，但不影响核心结论。 |
| **4-6** | 个别模糊 | 存在1-2处表述与素材存在张力，可能源于措辞不精确而非逻辑错误。 |
| **7-9** | 基本一致 | 推论与素材事实基本一致，仅个别细节可商榷。 |
| **10** | 完全一致 | 所有推论均可与素材事实逐条对应，无任何出入。 |

**上限锁定**：若报告的可检验推论占关键论断的比例 < **20%**，此项得分上限锁定为 **3 分**。

### 子维度2.2：因果方向错误（权重：0.30）

**操作性定义**：检测报告的因果归因方向是否与素材事实一致。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 严重倒置 | 将结果当作原因。 |
| **1-3** | 部分混淆 | 某条因果链方向存疑，或因果与相关关系未区分。 |
| **4-6** | 因果链模糊 | 因果方向大致正确但表述模糊，未明确方向性。 |
| **7-9** | 方向正确 | 因果方向全部正确，表述清晰。 |
| **10** | 精准且检验 | 不仅方向正确，还识别了素材中可验证因果方向的关键证据节点。 |

**上限锁定**：若报告未对核心问题做出任何明确的因果方向判断（仅描述相关、并列事实），此项得分上限锁定为 **3 分**。

### 子维度2.3：忽视决定性反证（权重：0.30）

**操作性定义**：检测报告是否忽略了素材中明确存在、且足以推翻或严重削弱其核心推论的信息。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 故意忽视 | 素材中存在明确可推翻报告核心推论的关键信息，报告完全未提及。 |
| **1-3** | 重大遗漏 | 遗漏了与推论构成明显张力的重要信息。 |
| **4-6** | 覆盖不足 | 触及了部分反证但处理不够充分，或遗漏了次要反证。 |
| **7-9** | 有效处理 | 对主要反证均有回应或解释，无重大遗漏。 |
| **10** | 主动吸纳 | 不仅处理了反证，还将反证纳入分析框架作为支撑或限定条件。 |

**上限锁定**：若报告因未做出足够明确的判断而系统性地“避免”了反证问题，此项得分上限锁定为 **3 分**。

**否决条件**：
- 发现任何推论与素材事实存在直接矛盾且未自行修正。备注：报告推论与素材事实存在直接冲突。
- 若该维度有3个及以上子维度得分 ≤ 3分，维度得分直接归零。备注：多个子维度表现极差，维度归零。

## 维度三：思维严谨程度（权重：0.10）

**操作性定义（全维度通用）**：思维严谨程度检测的是分析过程是否进行了多角度、有深度的思考。

### 子维度3.1：归因深度（权重：0.30）

**操作性定义**：检测报告的归因层次——是停留在现象描述和直接原因，还是追溯到了系统性/结构性的根本原因和预设条件。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 无归因 | 仅描述现象，未做任何归因。 |
| **1-3** | 浅层归因 | 归因于最表面的直接原因，未追问“为什么”。 |
| **4-6** | 中层归因 | 追溯了一到两层原因，触及行业或市场层面的解释。 |
| **7-9** | 结构归因 | 追溯到系统性/制度性/结构性的根本原因。 |
| **10** | 源头归因 | 揭示了现象得以发生的预设条件或基本假设。 |

### 子维度3.2：矛盾识别与处理（权重：0.25）

**操作性定义**：检测报告是否识别了素材中的矛盾信息，以及如何处理这些矛盾——是并列呈现、部分解释，还是将其纳入统一的分析框架解释成因。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 无视矛盾 | 素材中存在明显矛盾，报告未提及。 |
| **1-3** | 罗列矛盾 | 提到了矛盾但未尝试解释，仅并列呈现。 |
| **4-6** | 部分解释 | 对矛盾给出了解释，但解释不够深入或仅适用部分情况。 |
| **7-9** | 有效整合 | 对矛盾给出了合理解释，并将其纳入整体分析框架。 |
| **10** | 矛盾驱动洞见 | 以矛盾为分析起点，从中导出了超越素材的洞察。 |

### 子维度3.3：多视角审视（权重：0.25）

**操作性定义**：检测报告是否从多个利益相关方视角进行了分析，以及是否呈现了不同视角之间的冲突、博弈和信息不对称。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 单一视角 | 仅从一方视角分析。 |
| **1-3** | 两方视角 | 覆盖了两个利益相关方，但分析不对称。 |
| **4-6** | 多方罗列 | 覆盖了多个利益相关方，但分析并列，未呈现视角间的互动与冲突。 |
| **7-9** | 视角对话 | 呈现了不同视角之间的冲突与博弈，各自立场有合理解释。 |
| **10** | 视角超越 | 在呈现多方视角后，提出了超越各方立场的第三方分析位置。 |

### 子维度3.4：反模式意识（权重：0.20）

**操作性定义**：检测报告是否存在"伪深度"（用复杂术语包装空洞内容）或"防御性复杂"（因害怕遗漏不断添加边界条件导致结论不传递信息）。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 严重伪深度 | 用大量专业术语和复杂表述包装空洞内容。 |
| **1-3** | 明显伪深度 | 存在多处无实质内容的复杂表述。 |
| **4-6** | 偶有冗余 | 个别段落有冗余或过度修饰，但主体内容实在。 |
| **7-9** | 简洁有力 | 表述精准，无冗余，复杂性与问题匹配。 |
| **10** | 大道至简 | 用简明语言表达了深刻洞见，复杂被化解而非堆砌。 |

**否决条件**：
- 若3.1和3.2同时 ≤ 3分，维度得分直接归零。备注：报告未展现基本思考能力。
- 若该维度有3个及以上子维度得分 ≤ 3分，维度得分直接归零。备注：多个子维度表现极差，维度归零。

## 维度四：指令遵循度（权重：0.10）

**操作性定义（全维度通用）**：指令遵循度检测的是报告是否严格遵守了用户提示词中的每一项具体要求。

### 子维度4.1：任务覆盖度（权重：0.30）

**操作性定义**：检测报告是否回答了用户提出的所有问题，有无遗漏，各题处理深度是否均衡。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 完全未做 | 未回答任何提出的问题。 |
| **1-3** | 严重遗漏 | 遗漏了多个核心问题，或回答过于敷衍。 |
| **4-6** | 部分覆盖 | 回答了多数问题，但个别问题处理明显不足。 |
| **7-9** | 全面覆盖 | 回答了所有问题，各题处理深度均衡。 |
| **10** | 超额完成 | 不仅回答了所有问题，还在合理范围内超出了预期。 |

### 子维度4.2：格式遵循度（权重：0.25）

**操作性定义**：检测报告是否按用户要求的格式输出，结构是否清晰。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 格式错误 | 完全未按要求格式输出。 |
| **1-3** | 格式混乱 | 格式有重大偏差，影响可读性。 |
| **4-6** | 基本合规 | 格式大致正确，但有小偏差。 |
| **7-9** | 格式规范 | 格式符合要求，结构清晰。 |
| **10** | 格式典范 | 格式完全符合且美观易读。 |

### 子维度4.3：禁止事项遵守（权重：0.30）

**操作性定义**：检测报告是否遵守了"不搜索外部知识"、"不编造信源"、"仅基于给定素材分析"等禁止指令。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 严重违规 | 调用了外部搜索，或编造了信源。 |
| **1-3** | 可疑违规 | 存在疑似外部知识侵入的痕迹，但无法确定。 |
| **4-6** | 边界模糊 | 个别表述来源不明，但大概率是合理推断。 |
| **7-9** | 基本合规 | 未发现明显违规。 |
| **10** | 严格合规 | 可确认所有信息均来自素材或明确标注的推断。 |

### 子维度4.4：角色一致性（权重：0.15）

**操作性定义**：检测报告是否保持了要求的角色语气和分析立场。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 角色错位 | 完全未按要求的角色语气和立场输出。 |
| **1-3** | 角色漂移 | 多处偏离要求的分析角色。 |
| **4-6** | 偶有漂移 | 个别处语气不当，但主体保持。 |
| **7-9** | 角色稳定 | 全程保持要求的角色语气和立场。 |
| **10** | 角色驾驭 | 不仅保持角色，语气和立场的把握增强了分析的说服力。 |

**否决条件**：
- 维度得分 < 4.0。备注：报告未遵守基本任务指令。
- 若该维度有3个及以上子维度得分 ≤ 3分，维度得分直接归零。备注：多个子维度表现极差，维度归零。

## 维度五：素材引用度（权重：0.10）

**操作性定义（全维度通用）**：素材引用度检测的是报告中的关键论断是否有明确的素材支撑。

### 子维度5.1：关键论断锚定率（权重：0.35）

**操作性定义**：检测报告的核心结论是否能追溯到素材中的具体片段，锚定的证据是否构成完整逻辑链而非孤立单点支撑。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 无锚定 | 核心论断完全无素材支撑，凭空而来。 |
| **1-3** | 弱锚定 | 多数关键论断无明确素材对应。 |
| **4-6** | 部分锚定 | 部分关键论断有素材支撑，部分来源不明。 |
| **7-9** | 基本锚定 | 多数关键论断可追溯到素材中的具体片段。 |
| **10** | 完全锚定 | 每个关键论断均可精准定位至素材的具体段落，形成完整证据链。 |

### 子维度5.2：量化数据引用准确度（权重：0.30）

**操作性定义**：检测报告引用的数字是否与素材一致，有无数字捏造，是否保留了素材中的对比基数和限定条件。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 数据捏造 | 存在素材中没有的数字。 |
| **1-3** | 多处错误 | 引用数字与素材存在多处偏差。 |
| **4-6** | 个别偏差 | 个别数字与素材有出入（如近似值转述但未改变含义）。 |
| **7-9** | 数据准确 | 所有引用数字与素材一致。 |
| **10** | 精准且有上下文 | 数据引用不仅准确，还保留了素材中的对比基数和限定条件。 |

### 子维度5.3：信源标注透明度（权重：0.20）

**操作性定义**：检测报告引用时是否保留了素材中的信源身份，是否区分了素材信息与分析推断，信息层级是否清晰可辨。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 无任何标注 | 所有引用未标注来源，无法区分素材信息与分析推断。 |
| **1-3** | 偶有标注 | 极少数引用标注了来源，绝大多数没有。 |
| **4-6** | 部分标注 | 部分关键引用有信源标注，但不系统。 |
| **7-9** | 系统标注 | 主要引用均标注了信源身份，读者可区分。 |
| **10** | 完全可溯源 | 引用均附信源，推断均标注为推断，信息层级清晰可辨。 |

### 子维度5.4：无源推断密度（权重：0.15）

**操作性定义**：检测报告中无素材支撑的推断占总篇幅的比例。

| 分数 | 等级 | 行为描述（无素材支撑的推断占总篇幅比例） |
|:---|:---|:---|
| **0** | 全是无源推断 | 报告几乎全部是无素材支撑的猜测。 |
| **1-3** | 大量无源推断 | > 50%。 |
| **4-6** | 部分无源推断 | 20-50%。 |
| **7-9** | 少量推断已标注 | < 20%，且多数标注为推断。 |
| **10** | 推断极少且透明 | 仅在有充分推理链条时做推断，且全部明确标注。 |

**上限锁定**：若报告的无源推断密度为 0 或接近于 0（即报告几乎不做任何推断），且认知增量维度已被否决，则此项得分上限锁定为 **3 分**。

**否决条件**：若该维度有3个及以上子维度得分 ≤ 3分，维度得分直接归零。备注：多个子维度表现极差，维度归零。

---

## 维度六：边界偏移度（权重：0.10）

**操作性定义（全维度通用）**：边界偏移度检测的是报告是否引入了素材中不存在的外部事实信息并基于此展开推断。

### 子维度6.1：越界推断数量（权重：0.30）

**操作性定义**：检测报告是否引入了素材中不存在的事实性信息（实体名称、量化数据、具体事件、未在素材中有任何锚定点的因果关系）并基于此展开推断。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 大量越界 | 报告充斥着素材中毫无依据的事实性陈述。 |
| **1-3** | 多处越界 | 存在5处以上明显越界的推断。 |
| **4-6** | 偶有越界 | 存在2-5处越界推断。 |
| **7-9** | 极少越界 | 越界推断不超过2处，且影响不大。 |
| **10** | 严格守界 | 无一越界推断，所有陈述均有素材锚定或明确标注为推断。 |

**上限锁定**：若报告的越界推断数量为 0 或接近于 0，但同时对核心问题的回答未提供任何超越素材明说因果链的洞见，则此项得分上限锁定为 **3 分**。

### 子维度6.2：越界推断的标注率（权重：0.30）

**操作性定义**：检测报告是否将引入的外部事实信息或素材中不存在的因果关系推断标注为"推测/假设/推断"，以防止读者将其误认为素材事实。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 未标注 | 越界推断均未标注，被当作事实陈述。 |
| **1-3** | 极少标注 | 仅极少数越界推断有“可能”等弱化词，但未明确标注为推测。 |
| **4-6** | 部分标注 | 部分越界推断以“可能”表述，但未系统区分“事实”与“推测”。 |
| **7-9** | 基本标注 | 多数越界推断明确标注为推测、假设或推断。 |
| **10** | 完全标注 | 所有越界推断均明确标注其推断性质与不确定性级别，且在综合结论中未被当作事实使用。 |

**特别说明**：读者是否会合理地将这个陈述误认为是素材中的事实？如果会，则需要标注。如果读者显然知道这是分析者的概念工具，则不需要标注。
**注意**：若越界推断在标注后仍被综合结论当作事实使用，此项直接判 0 分。

### 子维度6.3：结论的绝对化程度（权重：0.20）

**操作性定义**：检测报告结论的措辞是否与素材支撑力匹配。区分"审慎有度"与"绝对化"。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 高度绝对化 | 大量使用“必然”、“一定”、“毫无疑问”等封闭式表述。 |
| **1-3** | 偏绝对化 | 多处结论以确定性口吻表述，但事实基础不足以支撑。 |
| **4-6** | 基本审慎 | 多数结论有“可能”、“大概率”等限定，但个别处过于绝对。 |
| **7-9** | 审慎有度 | 结论均有适当限定，确定性程度与素材支撑力匹配。 |
| **10** | 精准标度 | 不仅审慎，还对不同结论标注了置信度或概率判断，且与证据强度匹配。 |

**上限锁定**：若报告的所有结论均被“可能”、“一定程度上”等模糊限定词系统性消解，导致结论实际上不传递任何可检验的信息，则此项得分上限锁定为 **4 分**。

### 子维度6.4：不确定性声明质量（权重：0.20）

**操作性定义**：检测报告的不确定性声明是否贯穿全文、是否与具体论断对应、是否区分了不同不确定性来源和级别。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 无不确定性声明 | 全文无任何不确定性声明。 |
| **1-3** | 形式化声明 | 仅在开头或结尾有一句免责声明，与具体分析脱节。 |
| **4-6** | 局部声明 | 部分段落有不确定性声明，但不系统。 |
| **7-9** | 贯穿性声明 | 不确定性声明贯穿全文，与具体论断对应，明确区分了确定、不确定和可能确定三种状态。 |
| **10** | 结构化的不确定性 | 不仅贯穿全文，还对不确定性来源进行了分类，使读者清楚知道“哪里不确定、为什么不确定”。 |

**否决条件**：若该维度有3个及以上子维度得分 ≤ 3分，维度得分直接归零。备注：多个子维度表现极差，维度归零。

## 维度七：内部逻辑一致性（权重：0.10）

**操作性定义（全维度通用）**：内部逻辑一致性检测的是报告自身是否前后自洽，是否存在自相矛盾。

### 子维度7.1：前后矛盾数量（权重：0.30）

**操作性定义**：检测报告的核心结论之间是否存在直接冲突，或局部表述是否存在矛盾。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 核心矛盾 | 报告的核心结论之间存在直接冲突。 |
| **1-3** | 多处小矛盾 | 存在2处以上局部矛盾。 |
| **4-6** | 个别张力 | 存在1-2处表述有张力但可解释为措辞问题。 |
| **7-9** | 基本自洽 | 未发现明显矛盾，全文逻辑一致。 |
| **10** | 严格自洽 | 全文无任何前后不一致，且各部分的相互支撑关系清晰可辨。 |

**上限锁定**：若报告的各部分（如五个问题的回答）之间在逻辑上完全独立、不存在任何跨题引证或整合，则此项得分上限锁定为 **4 分**。

### 子维度7.2：因果链条完整性（权重：0.25）

**操作性定义**：检测报告的因果判断是否提供了从原因到结果的完整路径，关键中间环节是否均在。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 因果断裂 | 因果判断缺少关键中间环节。 |
| **1-3** | 多处跳跃 | 多处因果链条有明显跳跃。 |
| **4-6** | 基本完整 | 主要因果链条有中间环节，但个别处有跳跃或简化。 |
| **7-9** | 链条完整 | 从原因到结果的完整路径清晰，关键中间环节均在。 |
| **10** | 多路径因果 | 不仅完整，还区分了主路径与替代路径，呈现了因果网络而非单链条。 |

**上限锁定**：若报告未对核心问题构建从原因到结果的完整因果链条（仅做事实描述、相关陈述或复述素材已给出的因果），则此项得分上限锁定为 **4 分**。

### 子维度7.3：框架一致性（权重：0.25）

**操作性定义**：检测报告是否在开头提出了统一的分析框架，以及后续是否一致沿用。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 无框架或框架完全偏离 | 未提出框架，或开头提出框架但后续完全偏离。 |
| **1-3** | 框架偏离 | 开头提出了框架但后续大部分偏离。 |
| **4-6** | 框架松动 | 框架与后续分析有对应但不紧密。 |
| **7-9** | 框架一致 | 框架贯穿全文，各部分与框架的对应关系清晰。 |
| **10** | 框架迭代 | 框架不仅贯穿全文，还在分析过程中得到了细化或深化。 |

**注**：若报告未提出任何分析框架，此项评5分（中性）。

### 子维度7.4：综合结论与分点一致性（权重：0.20）

**操作性定义**：检测报告的最终综合结论是否与各分点分析相符，有无遗漏分点分析中的重要发现，有无添加分点分析中未论证的新结论。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 完全脱节 | 综合结论与各分点分析无关联，像是独立写的。 |
| **1-3** | 弱关联 | 综合结论仅涵盖部分分点，另一些分点的发现被遗漏。 |
| **4-6** | 基本一致 | 综合结论与分点大致一致，但存在个别不完全对应。 |
| **7-9** | 高度一致 | 综合结论准确总结了各分点分析的发现，无遗漏无添加。 |
| **10** | 涌现性总结 | 综合结论不仅总结了分点，还呈现了分点分析中各自无法看到的整体图景。 |

**上限锁定**：若报告未提供综合结论（各分题独立收尾、无任何跨题整合），此项直接判 **3 分**。

**否决条件**：
- 发现任何前后矛盾。备注：报告内部存在自相矛盾。
- 若该维度有3个及以上子维度得分 ≤ 3分，维度得分直接归零。备注：多个子维度表现极差，维度归零。

## 维度八：幻觉程度（权重：0.10）

**操作性定义（全维度通用）**：幻觉程度检测的是报告是否编造了素材中不存在的事实性信息。

### 子维度8.1：事实捏造数量（权重：0.40）

**操作性定义**：检测报告是否存在素材中完全没有、模型自行编造的人名、数字、日期、事件。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 严重捏造 | 存在核心事实的捏造。 |
| **1-3** | 多处捏造 | 存在3处以上非核心事实的捏造。 |
| **4-6** | 个别捏造 | 存在1-2处事实错误，可能是笔误而非故意捏造。 |
| **7-9** | 无捏造 | 经全文核对未发现事实捏造。 |
| **10** | 完全准确 | 不仅无捏造，所有事实性陈述均可与素材逐条对应验证。 |

### 子维度8.2：信源捏造数量（权重：0.35）

**操作性定义**：检测报告是否虚构了引用来源，或将素材中的信息错误归因到不存在的信源。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 捏造信源 | 虚构了引用来源。 |
| **1-3** | 信源嫁接 | 将素材中的信息错误归因到不存在的信源。 |
| **4-6** | 信源模糊 | 存在疑似虚构但无法确定的情况。 |
| **7-9** | 无捏造 | 未发现虚构的引用来源。 |
| **10** | 严格可溯 | 所有提及的信源均可在素材中找到对应。 |

### 子维度8.3：外推知识泄露（权重：0.25）

**操作性定义**：检测报告是否引入了素材中完全没有、且无法从素材合理推断的外部事实性信息（实体名称、量化数据、具体事件、无素材锚定点的因果关系）。核心区分：分析工具、方法论框架和理论概念的引入不属于外推信息泄漏。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 大量外部信息 | 报告中明显存在素材没有、且无法从素材合理推断的外部信息（实体、数据、事件、因果关系，分析工具、方法论框架和理论概念的引入不属于此项）。 |
| **1-3** | 多处泄露 | 存在3处以上疑似外部信息侵入。 |
| **4-6** | 个别可疑 | 存在1-2处疑似外部信息，但可能是常识性合理推断。 |
| **7-9** | 基本无泄露 | 未发现明显的外部信息侵入。 |
| **10** | 严格封闭 | 所有信息均可确认来自素材或从素材中合理推导，无任何外部信息痕迹。 |

**否决条件**：
- 发现任何事实捏造或信源捏造。备注：报告存在无中生有。
- 若该维度有3个及以上子维度得分 ≤ 3分，维度得分直接归零。备注：多个子维度表现极差，维度归零。

---

## 维度九：选择性偏差（权重：0.10）

**操作性定义（全维度通用）**：选择性偏差检测的是报告是否存在"片面引用"行为——是否仅引用支持自身结论的证据而忽视反例，是否覆盖了素材中呈现的多个利益相关方视角。

### 子维度9.1：反例引用与处理（权重：0.40）

**操作性定义**：检测报告是否引用了素材中与自身结论相悖的证据，以及如何处理这些反例——是并列呈现、部分处理，还是将反例整合进可检验的判断框架。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 完全忽视 | 素材中存在明确反例，报告完全未提及，且未解释原因。 |
| **1-3** | 选择性忽视 | 忽视了对结论构成明显挑战的重要反例。 |
| **4-6** | 部分处理 | 引用了部分反例，但处理不充分；或引用反例后未将其纳入分析框架。 |
| **7-9** | 有效处理 | 主要反例均有引用和合理处理，反例的存在增强了分析的信任度。 |
| **10** | 反例驱动 | 不仅处理了反例，还主动寻找反例来检验自身结论，将其作为分析深化的重要输入。 |

**上限锁定**：若报告对反例的“处理”仅是将素材中不同立场的信息并列呈现（“一方面…另一方面…”），而未将其整合进一个可检验的判断框架，则此项得分上限锁定为 **5 分**。

### 子维度9.2：证据代表性（权重：0.30）

**操作性定义**：检测报告所引用的证据是否代表了素材中的主流信息，还是只截取了符合特定立场的少数声音。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 严重偏倚 | 仅引用了极少数边缘性证据，完全不代表素材主流信息。 |
| **1-3** | 明显偏倚 | 引用偏重某一方或某一类证据，主流信息被忽视。 |
| **4-6** | 基本代表 | 引用的证据大致代表素材主流，但覆盖面有欠缺。 |
| **7-9** | 良好代表 | 证据选取覆盖了素材中的主要信息类型和立场。 |
| **10** | 精准代表 | 证据选取不仅代表主流，还体现了素材中的信息分布比例，未放大边缘声音。 |

### 子维度9.3：视角覆盖度（权重：0.30）

**操作性定义**：检测报告是否覆盖了素材中呈现的多个利益相关方视角，以及是否呈现了各方视角之间的信息不对称和认知差异。

| 分数 | 等级 | 行为描述 |
|:---|:---|:---|
| **0** | 单一视角 | 仅呈现一方视角，且未说明为何忽略其他方。 |
| **1-3** | 两方偏倚 | 覆盖两方视角但严重偏重一方。 |
| **4-6** | 多方但不均衡 | 覆盖了多个利益相关方，但分析深度不均。 |
| **7-9** | 多方均衡 | 素材中呈现的主要利益相关方均有覆盖，分析深度相当。 |
| **10** | 视角全息 | 不仅覆盖了素材中所有主要利益相关方，还呈现了各方视角之间的信息不对称和认知差异。 |

**否决条件**：若该维度有3个及以上子维度得分 ≤ 3分，维度得分直接归零。备注：多个子维度表现极差，维度归零。

## 权重分配与计分规则

| 维度 | 权重 | 
|:---|:---|
| 认知增量 | 0.20 |
| 外部逻辑一致性 | 0.10 |
| 思维严谨程度 | 0.10 |
| 指令遵循度 | 0.10 |
| 素材引用度 | 0.10 |
| 边界偏移度 | 0.10 |
| 内部逻辑一致性 | 0.10 |
| 幻觉程度 | 0.10 |
| 选择性偏差 | 0.10 |

**综合总分计算**：

第一步，计算原始综合总分：
> 原始综合总分 = 认知增量×0.20 + 外部逻辑一致性×0.10 + 思维严谨程度×0.10 + 指令遵循度×0.10 + 素材引用度×0.10 + 边界偏移度×0.10 + 内部逻辑一致性×0.10 + 幻觉程度×0.10 + 选择性偏差×0.10

第二步，应用否决折扣（如有）：
- 若**认知增量维度**触发否决（归零），则原始综合总分 **× 0.8**。
- 若**其他任一维度**触发否决（归零），则原始综合总分 **× 0.9**。
- 若同时存在认知增量否决与其他维度否决，按最低折扣执行，即 **× 0.8**。
- 若多个其他维度否决同时存在，折扣仍为 × 0.9（不叠加）。

最终综合总分 = 应用折扣后的原始综合总分（保留一位小数）。
